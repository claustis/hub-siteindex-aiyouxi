# hub-siteindex-aiyouxi

## 项目简介

这是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为静态页面的集合与发布渠道，方便使用者浏览和引用其中的内容。

## 目录结构

```
hub-siteindex-aiyouxi/
├── pages/           # 存放归档的 HTML 页面文件
├── assets/          # 页面引用的资源文件（如样式、脚本、图片等）
├── index.html       # 仓库首页导航页面
└── README.md        # 本文件
```

- `pages/` 目录下包含所有归档的独立 HTML 页面，每个页面为一个独立的文件或子目录。
- `assets/` 目录用于存放页面所依赖的公共资源，按需组织。
- `index.html` 提供简单的导航入口，便于浏览所有归档页面。

## 页面归档说明

本仓库中的 HTML 页面均为独立归档，彼此之间无关联。每个页面均以完整 HTML 文件形式存在，可直接在浏览器中打开运行，或通过 GitHub Pages 等静态托管服务进行访问。

归档页面可能包含以下类型：
- 信息展示页
- 功能演示页
- 工具页面
- 其他静态 HTML 内容

所有页面均不依赖外部服务或动态后端，仅使用前端技术实现。

## 使用方式

你可以直接通过以下方式使用本仓库中的页面：
- 克隆仓库到本地，在浏览器中打开任意 HTML 文件。
- 通过 GitHub Pages 服务访问在线版本（需在仓库设置中启用 Pages 功能）。
